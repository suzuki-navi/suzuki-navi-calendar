# suzuki-navi's calendar

suzuki-navi's calendar.

![Calendar Image](calendar.png)

## Installation

    $ gem specific_install -l https://github.com/suzuki-navi/suzuki-navi-calendar.git
    $ rbenv rehash

## Usage

    $ cal

    $ cal 2021

## Development

    $ bundle install
    $ bundle exec exe/cal
